## Description:
This is a 2D open-world inspired by the Swedish furniture store IKEA and the game Stardew Valley. The player navigates through the world as a cute shark plushie known as Blahaj. The goal is to explore the store and collect three iconic items, a 1KEA bag, a Djungelskog or bear plush, and a Blahaj or the shark plush. This project is focused on creating a fun experience using customized elements and has collision built in. It features two different environments, an interior and exterior with doors to enter and exit.

## Files
My project is built on several classes to maintain and separate my code into readable sections. I have a three classes: character, item, and shelf. The character class covers the shark sprite and its movements. The item class manages the items and how their collection is carried out and the shelf class handles the collision mechanics in the environment. The main function carries out all printing all assets onto the screen.

## Design Challenges
One of the more challenging design considerations woud be the implementation of image based text. Due to issues with the newer Python version, the text rendering was unable to load. I decided to bypass this by using images instead of text while using the Pillow library to load them in with transparent background. This decision helped resolve a technical hurdle while ensuring the typography remained true to my vision.
