# Title
Shopping at IKEA with Blahaj

## Repository
https://github.com/lenicole989/Final-Project-Open-World.git

## Description
A open-world based on IKEA and will include some furniture, stuffed animals, and 
shopping features. This project explores environmental storytelling using Blahaj as the
main character to find and interact with items, turning a familiar space into a digital
experience.

## Features
- Player Controller
	- When user presses movement keys the Blahaj will move across the screen
- Add to Cart
	- Shopping feature where Blahaj can "pick up" items and will be executed by
    touching the item or pressing a key
- Camera following
	- Making it appear like the user is following Blahaj through the store by moving the
    background and objects based on movement.

## Challenges
- Learning how to implement objects and have them stay at a position until user finds them
- Research how to animate Blahaj sprite based on which way it moves
- How to carry and drop an item

## Outcomes
Ideal Outcome:
- A explorable IKEA warehouse with multiple sections and several interactable plushies
  and objects. 

Minimal Viable Outcome:
- A functional showroom where the player can move Blahaj and successfully interact with
  an item.

## Milestones
- Week 1
  1. Set up the environment in VS code
  2. Create Blahaj sprite to match 2D movement

- Week 2
  1. Designing how the rooms will look like in IKEA and figuring out how to get the
     user to follow Blahaj
  2. Finding out how to program the pick up and drop interactions
  3. Set up boundaries to make sure Blahaj does not phase through shelves or walls
 
- Week N (Final)
  1. Make sure the sprites and items work as intended and the program runs smoothly
  2. Submit project
